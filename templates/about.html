about {{ site_name }}

tast.ie is a personal and optionally social bookmark repository inspired by early del.icio.us. it's a simple way to save, organize, and share links that matter to you or your community.

this application was written in large part because i had a few discussions about a real need for social/community bookmarks in current_year. it's self-hostable if you want, and easy to deploy because communities need to be able to easily spin up infrastructure that isn't owned by a company with financial incentives.

features

  • tagging – organize bookmarks with space-separated tags
  • full-text search – find what you're looking for in tags, titles, or descriptions
  • visibility controls – keep bookmarks public, visible to registered users only, or private

open source

tast.ie is open source software. you can view the code, report issues, or contribute on github.
