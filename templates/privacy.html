privacy

tast.ie is designed with simplicity and privacy in mind. here's what you should know about how your data is handled.

what is saved

  • account info – username, email, and a securely hashed password
  • bookmarks – URIs, titles, descriptions, and tags you save
  • timestamps – when bookmarks are created or updated

visibility controls

you control who sees your bookmarks. each bookmark can be set to:

  • public – visible to everyone
  • registered – visible only to logged-in users
  • private – visible only to you

data storage

all data is stored locally on the server running this instance.
